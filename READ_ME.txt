
Build Java backend:
Note: JRE 1.8 needed to run app

1) mvn clean install 
2) cd /sky-app/target
3) java -jar sky-app-0.0.1-SNAPSHOT.jar (Backend will be running on http://localhost:8080)

Build React frontend:

1) cd /sky-web
2) npm install
3) npm test (optional will run tests)
3) npm start (Frontend will run on http://localhost:3000)
